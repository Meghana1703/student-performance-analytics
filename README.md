# Student Analytics Dashboard 

A professional Python based web application designed to analyze and visualize academic performance trends using real world datasets.

##  Key Features
- **Data Filtering:** Interactive sidebar to filter by school and study habits.
- **Visual Analytics:** Dynamic Plotly charts showing score distributions.
- **Professional UI:** Custom dark mode interface styled with external CSS.

##  Tech Stack
- **Language:** Python 3.11
- **Libraries:** Pandas, Plotly, Streamlit
- **Styling:** Custom CSS3

##  How to Run Locally
1. Clone this repo.
2. Run `pip install -r requirements.txt`.
3. Run `streamlit run app.py`.
